Add People

Adds a public relevant person to the database, downloads the last two years of news published in The Guardian about him/her and calculate the sentiment score using NLPK and Blobtext

PUT /root/people/

Parameter Type Required Description
name str. Required Complete name of the public relevant person in English
country str. Required Nationality of the public relevant people expressed in 3 letter ISO code
party str. Optional Political party to which the person is affiliated

Sample Request

	{
		"name": "Liz Truss",
		"country": "GBR",
		"party": "Conservative Party"
	}

Sample Request in Phyton


	import requests
	dic = {
		'name': 'Liz Truss',
		'country': 'GBR',
		'party': 'Conservative Party'
		}
	response = requests.put(url, params=dic)
